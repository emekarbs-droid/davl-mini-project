# 🌸 Astra: Premium ML Dashboard

Astra is a high-end Machine Learning and Data Analytics dashboard built with Streamlit. It features a "Glassmorphism" aesthetic and provides a full suite of tools for data exploration, predictive modeling, and specialized sports analytics (optimized for IPL datasets).

## ✨ Key Features
* **Automated EDA:** Quick statistical summaries and correlation heatmaps.
* **Model Lab:** Train Regression or Classification models (Random Forest, Decision Trees, etc.) with custom hyperparameters.
* **Advanced Analytics:** 3D PCA visualization, K-Means Clustering, and Factor Analysis.
* **Player Corner:** Deep-dive ball-by-ball analysis for cricket/IPL datasets.
* **PDF Reporting:** Generate and download "Astra Strategic Reports" directly from the dashboard.
