# 📊 Weather Predictor & Data Analysis Web App

## 🚀 Overview
This project is an **interactive Streamlit web application** that allows users to upload datasets and perform:

- 📊 Exploratory Data Analysis (EDA)
- 📈 Data Visualization
- 🧠 Dimensionality Reduction (PCA & LDA)
- 🤖 Machine Learning (Regression & Classification)

It is designed to make **data analysis and prediction simple, fast, and interactive**.

---

## ✨ Features

### 📂 Data Upload & Preview
- Upload CSV datasets dynamically  
- View dataset preview instantly  

### 🧹 Data Preprocessing
- Handles missing values automatically  
- Encodes categorical variables  
- Standardizes numerical features  

### 📊 Data Visualization
- Correlation Heatmap  
- Histogram (Distribution Plot)  
- Box Plot  
- Scatter Plot  
- Pair Plot  

### 🧠 Dimensionality Reduction
- **PCA (Principal Component Analysis)**  
- **LDA (Linear Discriminant Analysis)**  

### 🤖 Machine Learning Models

#### 🔹 Regression
- Linear Regression  

#### 🔹 Classification
- Logistic Regression  
- Decision Tree  
- Random Forest  

### 📊 Model Evaluation
- Accuracy Score  
- Confusion Matrix  
- Model Comparison

## 📦 Installation & Setup

### 1️⃣ Clone the Repository

git clone https://github.com/siliconstardust/weather-predictor-streamlit.git
cd weather-predictor-streamlit

### 2️⃣ Install Dependencies
pip install -r requirements.txt

### 3️⃣ Run the App
streamlit run app.py
