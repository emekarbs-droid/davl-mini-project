COVID-19 Comprehensive Machine Learning Analysis Dashboard
📝 Overview

The COVID-19 ML Analysis Dashboard is an interactive web application built using Python and Streamlit. It serves as a comprehensive analytical tool that performs Exploratory Data Analysis (EDA) and applies multiple Machine Learning algorithms on real-world pandemic data.

The project aims to classify COVID-19 severity, predict death rates, and analyze global trends through an intuitive and visually engaging interface.

🚀 Key Features
📊 1. COVID Trend Analysis & EDA
Correlation heatmaps to study relationships between cases, deaths, and recoveries
PCA for dimensionality reduction and visualization of WHO region clusters
⚠️ 2. Pandemic Severity Classification
Classifies risk into Low, Medium, High
Models used: Logistic Regression, Decision Tree, Random Forest
Includes accuracy comparison and confusion matrix
📉 3. Death Rate Prediction
Uses Linear Regression
Predicts Deaths per 100 Cases
Evaluated using R² Score and Mean Squared Error
🌍 4. WHO Region Classification
Predicts region based on COVID statistics
Demonstrates ML capabilities and limitations
🏆 5. Model Comparison
Compares all models
Identifies the best-performing algorithm
🛠️ Tech Stack
Frontend: Streamlit (Custom UI)
Languages: Python
Libraries: Pandas, NumPy, Scikit-learn
Visualization: Matplotlib, Seaborn
🎯 Objective

To develop an interactive dashboard that combines data analysis, visualization, and machine learning to provide meaningful insights and predictions related to COVID-19.
