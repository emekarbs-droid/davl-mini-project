# Data Analysis Web Application

A full-stack data analysis web application built using a Flask backend and an HTML/CSS/JavaScript frontend. Upload your datasets securely via the web portal and receive complex, interactive Plotly visualizations across multiple tabs.

## Features
- **Frontend**: Clean UI with animated tabs, cards, and responsive layouts. No page reloads required across the dashboard.
- **Backend Functionality**: Features Data Preprocessing, Distribution Analysis (EDA), Statistics, PCA, linear Discriminant Analysis (LDA), and Factor Analysis (FA).
- **Interactive Visualizations**: Generates rich dynamic plots embedded securely into HTML pages seamlessly.
- **Automated Insights**: Uncovers high correlation pairs, constant features, missing data alerts, etc.

## Setup Instructions

Ensure you use Python 3.8+!

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open a browser and visit: `http://127.0.0.1:5000/`
