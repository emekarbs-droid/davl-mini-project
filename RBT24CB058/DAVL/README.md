# Data Analysis & Visualization Lab (DAVL)

A comprehensive Streamlit-based EDA application for comparing two datasets (Netflix vs Amazon Prime).

## Features
- Dataset Overview & KPI cards
- Data Quality analysis (missing values, duplicates, outliers)
- Preprocessing pipeline (impute, encode, scale)
- Univariate, Bivariate, Multivariate EDA
- Statistical tests (Shapiro-Wilk, descriptive stats)
- PCA, LDA, Factor Analysis
- Auto-generated insights
- Side-by-side dataset comparison

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
